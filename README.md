# Terraform Single Web Server

This project sets up a basic web server on AWS using Terraform.

## Steps

1. **Initialize**:
   ```bash
   terraform init
   ```

2. **Deploy**:
   ```bash
   terraform apply
   ```
   Confirm with `yes`.

3. **Access**:
   Terraform will output the server's public IP.

4. **Cleanup**:
   ```bash
   terraform destroy
   ```
